# CurseForge Hytale Tracking

## Overview
**CurseForge Hytale Tracking** is an application designed to track the top downloaded Hytale mods on CurseForge.

## Purpose
This project was created to gather data and monitor mod performance. It serves as an analytical tool to evaluate the success and reach of our development team, where I serve as a Main Developer and Admin.

## Features
* Tracks download statistics for Hytale mods on CurseForge.
* Provides data insights to see how well the team is performing compared to the top mods.
